# Detect deep nesting collapse (3+ levels)

Extend the existing 2-level nested wrapping if detection to handle 3+ levels of nesting that can be collapsed into a single guard clause.

**Bad:**

```javascript
function foo(a, b, c) {
  if (a) {
    if (b) {
      if (c) {
        doSomething()
        doMore()
      }
    }
  }
}
```

**Good:**

```javascript
function foo(a, b, c) {
  if (!a || !b || !c) return
  doSomething()
  doMore()
}
```

Should recursively collapse N levels and combine all conditions with || (after inversion). The 2-level case is already covered â€” this extends to deeper nesting.
